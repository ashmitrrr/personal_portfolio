## Features
- **Modern Tech Stack:** Built with semantic **HTML5** and custom **CSS3** (no heavy frameworks).
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **Animations:** CSS-only typing effects and scroll-triggered fade-ins for high performance.
- **Project Showcase:** dedicated sections for featured work like *SentimenTrade*.

## Built With
- **HTML5** - Structure & Semantics
- **CSS3** - Styling, Flexbox/Grid, and Animations
- **JavaScript** - minimal DOM manipulation for scroll observers
- **Google Fonts** - *Poppins*, *Inter*, and *Space Mono* typography
