## Features

- **Immersive 3D Background** — Live particle field built with Three.js and custom GLSL shaders, reacting to mouse movement
- **Custom Cursor** — Smooth trailing dot + ring cursor with hover state transitions
- **GSAP Animations** — Professional scroll-triggered reveals, staggered card entrances, and heading animations via GreenSock
- **3D Card Tilt** — Vanilla Tilt on all project and experience cards with glare effect
- **Rotating Typewriter** — Hero subtitle cycles through phrases with realistic typing/deleting speed
- **Animated Counters** — Stats count up on scroll into view
- **Tech Ticker** — Auto-scrolling marquee of technologies
- **Experience Timeline** — Dedicated section for KPMG, 180 Degrees Consulting, and CampChat
- **Responsive Design** — Fully optimised for Mobile, Tablet, and Desktop with hamburger menu
- **Noise Overlay** — Subtle film grain texture for depth and atmosphere

## Built With

- **HTML5** — Structure & Semantics
- **CSS3** — Custom properties, Flexbox/Grid, keyframe animations
- **JavaScript (Vanilla)** — Core logic, scroll observers, cursor, rotating text
- **Three.js** — WebGL particle background with custom vertex/fragment shaders
- **GSAP + ScrollTrigger** — Professional-grade scroll animations
- **Vanilla Tilt** — 3D perspective tilt on cards
- **Google Fonts** — *Syne*, *Cabinet Grotesk*, *DM Mono*
- **Font Awesome** — Icons

## Sections

- **Hero** — Name, rotating subtitle, CTA buttons, social links, animated profile image
- **About** — Bio, tech stack board (`// tech_stack.json`), animated stat counters
- **Experience** — Timeline: Certificates, Mentorship, 180 Degrees Consulting, CampChat
- **Projects** — CampChat, SentimenTrade, LayoverAI, AI Resume Analyzer, AI Expense Tracker etc (list might get updated time to time)
- **Contact** — Email CTA with glow orb effect
